# STranslate.Plugin.Translate.YoudaoLLM

基于有道大模型Pro版的翻译插件，适用于STranslate应用程序。

## 功能特性

- 支持有道大模型Pro版（14B模型）
- 通过handleOption参数选择Pro版本（值为0）
- 支持多种语言翻译
- 与STranslate无缝集成

## 安装

1. 下载`.spkg`文件
2. 在STranslate中通过"插件管理"导入插件

## 配置

1. 注册有道智云账号并开通大模型API
2. 获取App Key和App Secret
3. 在插件设置中填入凭证信息

## 使用方法

配置完成后，即可在STranslate中使用有道大模型Pro版进行翻译。

## 版本历史

- 1.0.0 初始版本，支持有道大模型Pro版翻译
